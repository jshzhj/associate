# associate

#### 描述
Go语言实现搜索引擎自动联想功能以及字符串搜索(敏感词过滤)

#### 安装

1.  git clone https://gitee.com/zhj-open-source/associate.git
2.  进入main目录,执行go build
3.  执行 ./associate
4.  浏览器访问http://localhost:8080/index

#### 说明

1.  Golang1.14版本(最新版本)实现,代码中注释非常详细,如有问题,欢迎指正
